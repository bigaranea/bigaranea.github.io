.Задача

|description
|Многострочный текст
|false

.Статус
|===
|*Атрибут* |*Ключ* |*Тип* |*Не пустой* |*Описание*

|Идентификатор статуса
|status_id
|Число
|true
|Идентификатор статуса задачи.

Является PK

|Наименование
|name
|Однострочный текст
|true
a|Статус исполнения задачи.

Одно из значений:

* 'todo' - к исполнению;
* 'in progress' - в работе;
* 'done' - исполнена;
* 'decined' - отменена.

По умолчанию 'todo'
