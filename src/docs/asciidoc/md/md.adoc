= Структура данных сервиса работы с задачами
Структура данных, используемая сервисом работы с задачами, приведена на диаграмме.

[plantuml, md, png]
----
include::md.puml[]
----

.Задача
|===
|*Атрибут* |*Ключ* |*Тип* |*Не пустой* |*Описание*

|Заголовок
|title
|Однострочнй текст
|true
|Краткое описание задачи.

Длина не более 255 символов

|Описание
|description
|Многострочный текст
|false
|Полное описание задачи, что необходимо сделать
|===

.Статус
|===
|*Атрибут* |*Ключ* |*Тип* |*Не пустой* |*Описание*

|Наименование
|name
|Однострочный текст
|true
a|Статус исполнения задачи.

Одно из значений:

* 'todo' - к исполнению;
* 'in progress' - в работе;
* 'done' - исполнена;
* 'decined' - отменена.

По умолчанию 'todo'
|===
